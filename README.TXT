======  mEbay - Iphone Theme ======

OVERVIEW
--------
This is standard Magento Iphone theme. You can easy to changes skin of theme, easy to customization.The theme works great on Apple, Android, Blackberry and Window Phones.


FEATURES
--------
+ Supports Iphone, Ipad, Blackberry, Android OS, and Windows Phone. 
+ Compatible with  browsers mobile (FireFox, Safari, Chrome,opera )
+ Optimized for fast loading
+ Easily customize with PSD files.
+ Support Magento 1.5.x, 1.6.x, 1.7.x
+ Easy to install, configure

INSTALLATION
------------

1. Copy /app, /skin & /js folder to magento location. 
2. Go to Admin/Configuration/Design, select Themes section:
+ Templates,  click Add Exception and enter info below:
	Matched Expression: iPhone|iPod|iPad|Android
	Value: mebay_mini
+ Skin (Images/CSS),  click Add Exception and enter info below:
	Matched Expression: iPhone|iPod|iPad|Android
	Value: mebay_mini 
+ Layout: 
	Matched Expression: iPhone|iPod|iPad|Android
	Value: mebay_mini

3. Save and go to front-end to see the theme. Note to clear the cache if you are enabling it.

ADDITIONAL EXTENSIONS
---------------------
This theme uses Easy Banner extension.
Please contact us to get the additional extension if you have purchased it.

If you buy Easy Banner extension:

Go to Admin/Easy Banner
	+ Go to Admin/Easy Banner/ Manage Banners and click into add banner button

		*Banner information:
			Identifier *: mEbay_mini
			Title *	    : Your banner title
			Show Title  : No
			Width (px) *: 100
			Height (px)*: 348
			Delay (ms) *: 2000
			Status	    :Enable
			
	+ Go to Admin/Easy Banner/ Manage Banner Items and click into add Banner Item.
		*Banner Item information:
			Banner*: mEbay_mini
			Content*:Banner item content
